# AlpaCSS

Alpa CSS is an opensource lite weight css framework. It's utility based css framework.

ALPA RULE:
[Breackpoint]:[Property]:[Value]

example 1:
In CSS-
display:block

In Alpa
sm:d:block
md:d:block

example 2:
In CSS-
text-align: center

In Alpa
sm:text:center
md:text:left
lg:text:right
